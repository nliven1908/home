Eye-Click

1

Objective

An unique take on teaching the art of photography.

2

Initial Project Planning

An end to end experience where the user will be able to learn the basics requirements to be a pro photographer.

3

Mechanics

This collaborative educational game would harbour stage based photography tutorials where the users will compete in real time by taking a photograph based on given instructions (apperture, composition, shutter speed, etc.) generated by the game algorithm.

4

Project Headline

Based on initial research there has only been one game developed by Cannon Labs on similar lines. (Though they only deal with lens configurations and making the user familar with the DSLR camera)
